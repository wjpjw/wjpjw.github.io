User Manual

Toolbox introduction

Clear: Clear the canvas and remove persistent objects in memory.

Selection Mode: Allow you to select an object and tweak its position, size, etc.

Configuration: Adjust the parameters for Langui Model.

Topple Arrow: Turn on/off arrow on a curve.

Undo: Remove the last object both from canvas and memory.


How to draw a circle

Draw a closed curve in 500 ms(configurable) in whatever shape!


How to draw a standard polygon(including triangle)

Draw a closed curve with roughly N sides for a N-sided standard polygon, which is pretty standard no matter how you spin or twist your drawing.


Special rectangles: diamonds

Try to start the rectangle with a -45/-135 degree initial angle.


How to create a text box

Click a place, once. Then enter the text. Finally hit enter key to submit.


How to draw a curve

Just follow your intuition. These curves will be smoothed. If you want to draw a raw curve without processing, you should draw slowly and make sure drawing time > 16s(configurable)
